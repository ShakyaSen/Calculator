
# Simple Calculator in Python

## Overview
This project is a basic calculator program written in Python.  
It allows users to perform simple arithmetic operations such as:

- Addition (`+`)
- Subtraction (`-`)
- Multiplication (`*`)
- Division (`/`)

The program takes user input, performs the selected operation, and displays the final result.

---

## Features
- User-friendly console interaction
- Supports four basic mathematical operations
- Handles division by zero errors
- Displays invalid operator messages
- Beginner-friendly Python project

---

## Technologies Used
- Python 3.14

---

## How the Program Works
1. The program displays available operations.
2. The user enters:
   - First number
   - Operator
   - Second number
3. The program checks the operator using `if-elif` conditions.
4. The calculation is performed.
5. The result is displayed on the screen.

---

## Code Concepts Used
This project demonstrates the use of:

- Variables
- User input with `input()`
- Type conversion using `float()`
- Conditional statements (`if`, `elif`, `else`)
- Arithmetic operators
- Error handling for division by zero
